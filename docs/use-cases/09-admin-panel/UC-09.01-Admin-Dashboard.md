# Use Case-09.01: Admin Dashboard Overview

**Use Case ID:** UC-09.01  
**Module:** Admin Panel  
**Actor chính:** Admin  
**Actor phụ:** Không có

**Mô tả ngắn:** Admin xem dashboard tổng quan với các metrics và thống kê quan trọng

**Trigger:** Admin đăng nhập vào admin panel hoặc nhấn "Dashboard"

**Pre-condition:**

- Admin đã đăng nhập
- Admin có quyền truy cập admin panel

**Post-condition:**

- Dashboard hiển thị đầy đủ metrics và biểu đồ

**Main Flow:**

1. Admin đăng nhập vào admin panel
2. Hệ thống chuyển hướng đến Dashboard page
3. Hệ thống hiển thị các thẻ metrics chính:
   - Tổng số người dùng (học viên + giảng viên)
   - Tổng số khóa học (đã duyệt + chờ duyệt)
   - Tổng doanh thu
   - Người dùng hoạt động (7 ngày gần nhất)
4. Hệ thống hiển thị chỉ số tăng trưởng:
   - % thay đổi so với kỳ trước (tăng/giảm)
   - Icon và màu sắc (xanh: tăng, đỏ: giảm)
5. Hệ thống hiển thị biểu đồ:
   - Biểu đồ đường: Tăng trưởng người dùng (theo tháng)
   - Biểu đồ cột: Doanh thu (theo tháng)
   - Biểu đồ đường: Xu hướng tạo khóa học
6. Hệ thống hiển thị hoạt động gần đây:
   - 5 đăng ký mới nhất
   - 5 khóa học được gửi mới nhất
   - 5 thanh toán mới nhất
7. Hệ thống hiển thị quick actions:
   - "Duyệt khóa học chờ duyệt" (số badge)
   - "Xem đơn giảng viên" (số badge)
   - "Xem báo cáo" (số badge)
8. Hệ thống hiển thị time range filter:
   - Dropdown: "7 ngày", "30 ngày", "3 tháng", "Tùy chỉnh"
   - Mặc định: "7 ngày"
9. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Admin thay đổi time range**

  - Tại bước 8: Admin chọn time range khác
  - Hệ thống reload data theo time range mới
  - Cập nhật metrics, biểu đồ, chỉ số tăng trưởng
  - Tiếp tục bước 9

- **AF2: Admin chọn Custom time range**

  - Tại bước 8: Admin chọn "Tùy chỉnh"
  - Hệ thống hiển thị date picker (from - to)
  - Admin chọn khoảng thời gian
  - Hệ thống validate: from date < to date
  - Load data theo custom range
  - Tiếp tục bước 9

- **AF3: Admin xuất báo cáo PDF**

  - Tại bước 3: Admin nhấn "Export PDF"
  - Hệ thống generate PDF report:
    - Chứa tất cả metrics
    - Chứa biểu đồ (images)
    - Thông tin time range
  - Download file: "Dashboard*Report*[Date].pdf"

- **AF4: Admin click quick action**
  - Tại bước 7: Admin nhấn một quick action
  - "Duyệt khóa học" → Navigate đến UC-09.03
  - "Xem đơn giảng viên" → Navigate đến UC-09.04
  - "Xem báo cáo" → Navigate đến UC-09.05
