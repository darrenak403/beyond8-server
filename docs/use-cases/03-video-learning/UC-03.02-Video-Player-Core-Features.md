# Use Case-03.02: Video Player Core Features

**Use Case ID:** UC-03.02

**Module:** Học tập qua Video (Video Learning)

**Actor chính:** Khách/Học viên (Guest/Student)

**Actor phụ:** AWS CloudFront, React Player, HLS/DASH Player

**Mô tả ngắn:** Hệ thống cung cấp video player với các tính năng cơ bản: play/pause, seek, quality selection, fullscreen

**Trigger:** User click vào video lesson hoặc video được tự động tải

**Pre-condition:**

- Lesson có video đã được transcoded thành công
- Video URL từ CloudFront có sẵn
- User có quyền truy cập video (enrolled hoặc preview video)

**Post-condition:**

- Video player được load thành công
- User có thể phát, tạm dừng, tua, và điều chỉnh chất lượng
- Player hiển thị đúng thông tin (duration, current time)

**Main Flow:**

1. User click vào lesson hoặc video tự động load
2. Hệ thống kiểm tra quyền truy cập:
   - Enrolled student: load với signed URL
   - Guest preview: load với public URL
3. Hệ thống tải HLS manifest từ CloudFront
4. React Player khởi tạo với HLS source
5. Player hiển thị video controls:
   - **Play/Pause button** (spacebar hoặc click)
   - **Progress bar** với seek functionality
   - **Duration display** (total time)
   - **Current time display** (elapsed time)
   - **Volume control** (0-100%)
   - **Quality selector** (360p, 480p, 720p, 1080p, Auto)
   - **Fullscreen button**
   - **Settings menu** (speed: 0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x)
   - **Subtitles** (nếu có transcript)
6. Player load video trong < 3 giây (từ CloudFront CDN)
7. Adaptive bitrate streaming:
   - Hệ thống detect bandwidth user
   - Tự động chọn quality phù hợp
   - Auto-adjust nếu network thay đổi
8. User có các tương tác:
   - **Play/Pause**: Click button hoặc spacebar
   - **Seek**: Click hoặc drag progress bar
   - **Volume**: Adjust slider
   - **Quality**: Select từ dropdown
   - **Speed**: Select từ dropdown
   - **Fullscreen**: Click icon hoặc press F
   - **Subtitle**: Toggle on/off (nếu có)
9. Kết thúc Use Case khi user thoát video

**Alternate Flow:**

- **AF1: User truy cập từ mobile device**

  1. Player hiển thị mobile-optimized controls
  2. Fullscreen mặc định ở landscape mode
  3. Touch controls: tap để play/pause, swipe để seek
  4. Quay lại main flow bước 7

- **AF2: Network bandwidth thấp**

  1. Hệ thống tự động downgrade quality xuống 360p
  2. Hiển thị notification: "Adjusting quality for your connection"
  3. Continue playback
  4. Quay lại main flow bước 8

- **AF3: User muốn xem lại từ vị trí cũ**
  1. Hệ thống load saved position từ database (progress tracking)
  2. Player seek tới vị trí đó tự động
  3. Quay lại main flow bước 8

**Exception Handling:**

- **EH1: Network lỗi trong quá trình phát video**

  - Player pause tự động
  - Hiển thị error message: "Network error. Retrying..."
  - Auto-retry load từ CloudFront
  - Nếu fail 3 lần, hiển thị: "Unable to load video. Please refresh."

- **EH2: Video codec không được hỗ trợ**

  - Fallback tới progressive download (không streaming)
  - Hiển thị warning: "Your browser may not support this video format"

- **EH3: Signed URL hết hạn (student video)**
  - Player pause khi URL hết hạn (24 giờ)
  - Gửi request mới để lấy signed URL mới
  - Resume playback tự động

**Business Rules:**

- BR-04: Access control - chỉ enrolled students hoặc preview video được phép xem
- BR-06: Video limits - hỗ trợ tối đa 4K (3840x2160)
- Guest preview video: có watermark PREVIEW ở góc (xem UC-03.03)
- Student video: KHÔNG có watermark

**Non-Functional Requirements:**

- **Performance:**
  - Video load time < 3 giây
  - Playback smooth trên 3G connection (tối thiểu 480p)
  - Duration support: tối đa 30 phút (theo BR-06)
  - Seek response < 500ms
- **Compatibility:**
  - Modern browsers: Chrome, Firefox, Safari, Edge
  - Mobile: iOS Safari, Chrome Android
  - Desktop: Windows, macOS, Linux
- **Features:**
  - Adaptive bitrate: 360p, 480p, 720p, 1080p (tối đa, phù hợp với BR-06)
  - Codec: H.264 (primary), VP9 (optional)
  - Streaming protocol: HLS (HTTP Live Streaming) cho adaptive playback
  - Closed captions: VTT format (nếu có transcript)
- **Security:**
  - Disable right-click context menu
  - Disable developer tools video download
  - Signed URLs cho enrolled students (expire 24h)
  - HTTPS only

**Related Use Cases:**

- UC-03.01: Upload Video (source video)
- UC-03.03: Guest Preview Mode (với watermark)
- UC-03.04: Student Watch Mode (tracking progress)
- UC-06.01: Auto Update Video Progress (polling)

**Postcondition Details:**

- Player state được lưu (current position, quality, volume)
- Progress tracking trigger (UC-06.01) nếu user enrolled
- Watermark (UC-03.03) applied nếu user là guest preview
