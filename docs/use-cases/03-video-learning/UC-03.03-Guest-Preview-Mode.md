# Use Case-03.03: Guest Preview Mode

**Use Case ID:** UC-03.03

**Module:** Học tập qua Video (Video Learning)

**Actor chính:** Khách (Guest User)

**Actor phụ:** Không có

**Mô tả ngắn:** Guest có thể xem tối đa 3 preview videos per course

**Trigger:** Guest user nhấn vào lesson có badge PREVIEW

**Pre-condition:**

- Lesson được instructor đánh dấu is_preview = true
- Khóa học có tối đa 3 preview videos

**Post-condition:**

- Guest xem được video với watermark PREVIEW
- Hiển thị banner kêu gọi đăng ký khóa học
- Không track progress

**Main Flow:**

1. Guest user duyệt course curriculum
2. Hệ thống hiển thị badge PREVIEW bên cạnh lessons được đánh dấu preview
3. Guest click vào lesson có badge PREVIEW
4. Hệ thống load React Player với:

   - Video YouTube ID
   - Watermark "PREVIEW" ở góc phải (opacity 30%)

5. Hệ thống hiển thị banner dưới player:

   - Text: "Đăng ký khóa học để xem toàn bộ nội dung"
   - Button: "Enroll Now" → link đến course enrollment

6. Hệ thống disable right-click trên player
7. Hệ thống KHÔNG track watch progress cho Guest
8. Hệ thống KHÔNG hiển thị button "Mark as Complete"
9. Guest xem video
10. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Instructor đánh dấu lesson làm preview**

  - Instructor vào curriculum editor
  - Click checkbox "Preview" bên cạnh lesson
  - Hệ thống kiểm tra: số lượng preview videos hiện tại < 3
  - Nếu < 3: update lesson.is_preview = true
  - Nếu = 3: hiển thị error "Bạn chỉ có thể đánh dấu tối đa 3 video làm preview. Vui lòng bỏ đánh dấu video khác trước."
  - Hiển thị badge PREVIEW trong curriculum

- **AF2: Instructor bỏ đánh dấu preview**

  - Instructor uncheck checkbox "Preview"
  - Hệ thống update lesson.is_preview = false
  - Remove badge PREVIEW
  - Enable lại checkbox cho các lessons khác (nếu đang bị disable)

- **AF3: Guest click "Enroll Now"**
  - Tại bước 5: Guest nhấn button "Enroll Now"
  - Hệ thống chuyển hướng đến course detail page
  - Hiển thị enrollment options (Free hoặc Add to Cart)
  - Kết thúc Use Case
