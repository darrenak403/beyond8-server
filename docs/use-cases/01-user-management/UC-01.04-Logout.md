# Use Case-01.04: Logout

**Use Case ID:** UC-01.04

**Module:** User Management

**Actor chính:** Học Viên, Giảng Viên, Quản Trị Viên

**Actor phụ:** Không có

**Mô tả ngắn:** User kết thúc phiên làm việc (session)

**Trigger:** User nhấn nút "Logout"

**Pre-condition:**

- User đã đăng nhập

**Post-condition:**

- JWT token bị vô hiệu hóa
- User được chuyển hướng đến trang chủ

**Main Flow:**

1. User nhấn nút "Logout" (có sẵn trong menu header)
2. Hệ thống vô hiệu hóa JWT token (xóa khỏi client)
3. Hệ thống xóa session storage phía client
4. Hệ thống chuyển hướng user đến trang chủ
5. Kết thúc Use Case

**Alternate Flow:** Không có

**Business Rules:** Không có quy tắc cụ thể

**Requirement Traceability:**

- REQ-01.04
