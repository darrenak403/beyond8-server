# Use Case-04.10: Chấm Điểm Hỗ Trợ AI

**Use Case ID:** UC-04.10  
**Module:** Hệ Thống Quiz & Assignment  
**Actor chính:** System (Gemini Flash 2.5)  
**Actor phụ:** Instructor (xem lại nếu confidence thấp)

**Mô tả ngắn:** AI tự động chấm điểm câu tự luận hoặc assignment

**Trigger:** Student nộp câu tự luận hoặc assignment với grading method = AI-Assisted

**Pre-condition:**

- Submission đã được lưu
- Rubric chấm điểm tồn tại

**Post-condition:**

- Điểm AI được lưu
- Feedback được tạo
- Instructor nhận thông báo nếu confidence thấp

**Main Flow:**

1. Hệ thống kích hoạt AI grading process (queue)
2. Hệ thống chuẩn bị payload:
   - Question/Assignment prompt
   - Student answer
   - Grading rubric
   - Max points
3. Hệ thống gửi request đến Gemini API
4. Gemini xử lý và trả về:
   - Score (0 - max points)
   - Feedback (text)
   - Confidence (0-100%)
5. Hệ thống lưu kết quả:
   - ai_score
   - ai_feedback
   - confidence
6. Nếu confidence ≥ 70%:
   - Đánh dấu status = "Graded (AI)"
   - Student nhận notification với score và feedback
7. Nếu confidence < 70%:
   - Đánh dấu status = "Pending Review"
   - Flag để Instructor xem lại
   - Instructor nhận notification
8. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Lỗi Gemini API**

  - Tại bước 3: API timeout hoặc error
  - Hệ thống retry 3 lần với exponential backoff
  - Nếu vẫn thất bại:
    - Đánh dấu status = "Pending Manual Grading"
    - Thông báo Instructor
    - Kết thúc Use Case

- **AF2: Thời gian xử lý > 10 giây**

  - Tại bước 4: API phản hồi chậm
  - Hệ thống ghi log warning
  - Tiếp tục chờ (tối đa 30 giây)
  - Nếu > 30 giây → Chuyển sang AF1

- **AF3: Instructor xem lại điểm AI**
  - Sau bước 7: Instructor không đồng ý điểm AI
  - Kích hoạt UC-04.11 (Manual Grading)
  - Instructor override điểm AI
  - Điểm AI vẫn được lưu để đối chiếu
