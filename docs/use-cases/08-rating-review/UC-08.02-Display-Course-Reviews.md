# Use Case-08.02: Display Course Reviews

**Use Case ID:** UC-08.02  
**Module:** Rating & Review  
**Actor chính:** Guest, Student, Instructor  
**Actor phụ:** Không có

**Mô tả ngắn:** Hiển thị reviews và ratings trên course detail page

**Trigger:** User truy cập trang course detail

**Pre-condition:**

- Course tồn tại
- Course đã được publish

**Post-condition:**

- Reviews và rating summary được hiển thị

**Main Flow:**

1. User điều hướng đến course detail page
2. Hệ thống hiển thị phần Rating Summary:
   - Đánh giá trung bình (1 chữ số thập phân, ví dụ: 4.3/5.0)
   - Tổng số reviews (ví dụ: "Based on 234 reviews")
   - Biểu đồ phân bố sao:
     - 5 sao: x% (bar chart)
     - 4 sao: y%
     - 3 sao: z%
     - 2 sao: w%
     - 1 sao: v%
3. Hệ thống hiển thị danh sách reviews:
   - Avatar student
   - Tên student
   - Rating (sao)
   - Văn bản review
   - Số lượt hữu ích ("X người thấy hữu ích")
   - Ngày tạo (relative time: "2 days ago")
4. Hệ thống hiển thị bộ lọc và sắp xếp:
   - Dropdown sắp xếp: "Gần đây nhất", "Đánh giá cao nhất", "Đánh giá thấp nhất", "Hữu ích nhất"
   - Filter rating: "Tất cả", "5 sao", "4 sao", "3 sao", "2 sao", "1 sao"
5. Hệ thống hiển thị 10 reviews đầu tiên (pagination)
6. Hệ thống hiển thị button "Load More" hoặc pagination controls
7. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Không có reviews**

  - Tại bước 2: Course chưa có review nào
  - Hiển thị: "Chưa có đánh giá cho khóa học này"
  - Không hiển thị rating summary và biểu đồ
  - Kết thúc Use Case

- **AF2: User áp dụng filter**

  - Tại bước 4: User chọn filter (ví dụ: "5 sao")
  - Hệ thống lọc chỉ hiển thị reviews có rating = 5
  - Cập nhật số lượng reviews hiển thị
  - Reset về trang 1

- **AF3: User thay đổi sắp xếp**

  - Tại bước 4: User chọn sort option (ví dụ: "Hữu ích nhất")
  - Hệ thống sắp xếp lại reviews theo helpful_count giảm dần
  - Refresh danh sách

- **AF4: User nhấn "Load More"**

  - Tại bước 6: User nhấn "Load More"
  - Hệ thống load thêm 10 reviews tiếp theo
  - Append vào danh sách hiện tại
  - Nếu không còn reviews: ẩn button "Load More"

- **AF5: Pagination**
  - Tại bước 6: User nhấn page number hoặc Next/Previous
  - Hệ thống load trang tương ứng
  - Scroll to top của review section
