# Use Case-08.05: Course Rating Calculation

**Use Case ID:** UC-08.05  
**Module:** Rating & Review  
**Actor chính:** System  
**Actor phụ:** Không có

**Mô tả ngắn:** Hệ thống tự động tính toán và cập nhật rating trung bình của course

**Trigger:**

- Student submit review mới (UC-08.01)
- Student chỉnh sửa review (UC-08.01 AF4)
- Admin xóa review (UC-08.04 AF3)

**Pre-condition:**

- Course tồn tại
- Có thay đổi về reviews (thêm/sửa/xóa)

**Post-condition:**

- Average rating được cập nhật
- Cache được invalidate
- Rating mới hiển thị trên course card và detail page

**Main Flow:**

1. Hệ thống nhận trigger từ review operation
2. Hệ thống query tất cả reviews active của course:
   - WHERE course_id = X AND status != 'deleted'
3. Hệ thống tính toán:
   - total_reviews = COUNT(reviews)
   - sum_ratings = SUM(rating)
   - average_rating = sum_ratings / total_reviews
4. Hệ thống làm tròn average_rating đến 1 chữ số thập phân
5. Hệ thống cập nhật bảng courses:
   - average_rating
   - total_reviews
   - updated_at
6. Hệ thống invalidate cache:
   - Cache key: "course:{course_id}:rating"
   - Xóa cache entry
7. Hệ thống cập nhật course trong search index (Elasticsearch/Algolia):
   - Để hỗ trợ sort by rating
8. Hệ thống trigger UI update (real-time nếu có WebSocket):
   - Broadcast rating update event
9. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Course không có reviews**

  - Tại bước 2: Không có review nào (hoặc tất cả đã bị xóa)
  - Đặt average_rating = NULL hoặc 0
  - Đặt total_reviews = 0
  - Tiếp tục bước 5

- **AF2: Background calculation**

  - Tại bước 1: Trigger từ scheduled job (hàng ngày)
  - Hệ thống recalculate rating cho tất cả courses
  - Đảm bảo data consistency
  - Sửa sai số nếu có

- **AF3: Cache update**

  - Sau bước 6: Hệ thống warm up cache
  - Tính toán rating mới
  - Lưu vào cache với TTL = 24 giờ
  - Cache sẽ được dùng cho course listing và search

- **AF4: Rating distribution calculation**
  - Tại bước 3: Hệ thống tính thêm phân bố rating
  - five_star_count = COUNT(rating = 5)
  - four_star_count = COUNT(rating = 4)
  - three_star_count = COUNT(rating = 3)
  - two_star_count = COUNT(rating = 2)
  - one_star_count = COUNT(rating = 1)
  - Tính phần trăm cho mỗi rating level
  - Lưu vào bảng course_rating_stats
