# Use Case-06.03: Course Progress Dashboard

**Use Case ID:** UC-06.03

**Module:** Progress Tracking

**Actor chính:** Student

**Actor phụ:** Không có

**Mô tả ngắn:** Hiển thị tổng quan tiến độ học tập của Student

**Trigger:** Student truy cập My Courses dashboard

**Pre-condition:**

- Student đã đăng nhập

**Post-condition:**

- Danh sách courses đã enroll được hiển thị với tiến độ
- Student có thể filter và sort courses

**Main Flow:**

1. Student navigate đến Student Dashboard hoặc click "My Courses"
2. Hệ thống fetch danh sách enrollments của student
3. Hệ thống tính toán thống kê tổng quan:
   - **Tổng số courses đã đăng ký:** Count(enrollments)
   - **Tổng số courses hoàn thành:** Count(enrollments where progress = 100%)
   - **Tổng số giờ học:** Sum(time spent trên tất cả courses)
4. Hệ thống hiển thị 3 cards thống kê ở bước 3
5. Hệ thống hiển thị danh sách courses dưới dạng cards:
   - **Course thumbnail/cover image**
   - **Tiêu đề course**
   - **Tên instructor**
   - **Progress bar với phần trăm** (ví dụ: 45%)
   - **Số lessons hoàn thành / Tổng số lessons** (ví dụ: 12/25 lessons)
   - **Timestamp truy cập lần cuối** (ví dụ: "2 giờ trước", "3 ngày trước")
   - **Button "Continue Learning"**
6. Hệ thống hiển thị bộ lọc:
   - Dropdown filter:
     - "Tất cả"
     - "Đang học" (0% < progress < 100%)
     - "Đã hoàn thành" (progress = 100%)
     - "Chưa bắt đầu" (progress = 0%)
7. Hệ thống hiển thị tùy chọn sắp xếp:
   - Dropdown sort:
     - "Truy cập gần đây" (mặc định)
     - "Tiến độ (cao đến thấp)"
     - "Tiến độ (thấp đến cao)"
     - "Ngày đăng ký (mới nhất)"
     - "Ngày đăng ký (cũ nhất)"
8. Student có thể:
   - Click vào course card → chuyển đến course detail page
   - Click "Continue Learning" → chuyển đến lesson đang học hoặc lesson tiếp theo
   - Filter courses theo status
   - Sort courses theo tiêu chí
9. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Student chưa enroll course nào**

  - Tại bước 2: Không có enrollments
  - Hệ thống hiển thị:
    - Thống kê tổng quan: tất cả = 0
    - Empty state: "Bạn chưa đăng ký course nào"
    - Button "Browse Courses" → chuyển đến course catalog
  - Kết thúc Use Case

- **AF2: Student filter courses**

  - Tại bước 6: Student chọn filter (ví dụ: "Đang học")
  - Hệ thống reload danh sách chỉ hiển thị courses có 0% < progress < 100%
  - Cập nhật số lượng courses hiển thị
  - Kết thúc Use Case

- **AF3: Student sort courses**

  - Tại bước 7: Student chọn sort (ví dụ: "Tiến độ cao đến thấp")
  - Hệ thống sắp xếp lại courses theo progress_percentage DESC
  - Reload danh sách với thứ tự mới
  - Kết thúc Use Case

- **AF4: Student click "Continue Learning"**

  - Tại bước 8: Student click "Continue Learning" trên một course
  - Hệ thống xác định lesson tiếp theo:
    - Nếu có video đang xem dở (progress < 95%): tiếp tục video đó
    - Nếu không: tìm lesson chưa hoàn thành đầu tiên theo thứ tự
  - Chuyển hướng đến video player hoặc quiz/assignment page
  - Kết thúc Use Case

- **AF5: Student click vào course card**

  - Tại bước 8: Student click vào course thumbnail/title
  - Chuyển hướng đến course detail page với section progress breakdown (UC-06.04)
  - Kết thúc Use Case

- **AF6: Course vừa hoàn thành 100%**
  - Tại bước 5: progress_percentage = 100% (mới đạt)
  - Hệ thống hiển thị badge "Completed" trên course card
  - Hiển thị congratulations banner: "Chúc mừng! Bạn đã hoàn thành [Course Name]"
  - Nếu có certificate: Button "Download Certificate"
  - Kết thúc Use Case
