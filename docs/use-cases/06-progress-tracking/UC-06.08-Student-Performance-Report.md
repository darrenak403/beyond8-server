# Use Case-06.08: Student Performance Report

**Use Case ID:** UC-06.08

**Module:** Progress Tracking

**Actor chính:** Student

**Actor phụ:** Không có

**Mô tả ngắn:** Student xem performance report của bản thân

**Trigger:** Student truy cập Performance Report trong profile

**Pre-condition:**

- Student đã đăng nhập
- Student đã enroll ít nhất 1 course

**Post-condition:**

- Performance report được hiển thị
- Student hiểu rõ điểm mạnh/yếu của mình
- Gợi ý courses được đề xuất

**Main Flow:**

1. Student navigate đến Student Profile
2. Student click tab "Performance Report" hoặc "My Progress"
3. Hệ thống fetch toàn bộ learning data của student:
   - Enrollments và progress
   - Quiz/Assignment scores
   - Time spent on courses
   - Completion history
4. Hệ thống tính toán thống kê tổng quan:
   - **Tổng số courses đã đăng ký:** Count(enrollments)
   - **Courses đã hoàn thành:** Count(progress = 100%)
   - **Tổng số giờ học:** Sum(time_spent) từ tất cả courses
   - **Điểm quiz trung bình:** Average(quiz scores)
   - **Điểm assignment trung bình:** Average(assignment scores)
5. Hệ thống hiển thị Performance Dashboard:

   **A. Thống kê tổng quan:**

   - 5 cards hiển thị metrics ở bước 4

   **B. Bảng phân tích theo course:**

   - Table columns:
     - Tên course
     - Tiến độ (%)
     - Điểm TB quizzes
     - Điểm TB assignments
     - Thời gian đã dành (hours)
     - Status (Đang học/Hoàn thành)
   - Sort và filter khả dụng

   **C. Biểu đồ hiệu suất:**

   - Line chart: Điểm quiz/assignment theo thời gian
   - X-axis: Thời gian (tháng)
   - Y-axis: Điểm trung bình
   - 2 lines: Quiz scores, Assignment scores

   **D. Điểm mạnh và điểm yếu:**

   - **Điểm mạnh (Strengths):**
     - Top 3 chủ đề/courses có điểm cao nhất
     - Icon ✓ màu xanh
   - **Điểm yếu (Weaknesses):**
     - Top 3 chủ đề/courses có điểm thấp nhất
     - Icon ⚠️ màu vàng
     - Gợi ý: "Cần ôn tập thêm"

   **E. Gợi ý courses:**

   - "Dành cho bạn" - Recommended courses
   - Dựa trên:
     - Courses đã hoàn thành
     - Chủ đề yêu thích
     - Điểm yếu cần cải thiện
   - Hiển thị 5 courses recommended

6. Student có thể:
   - Xem chi tiết từng course
   - Filter report theo khoảng thời gian
   - Export report
7. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Student chưa có data**

  - Tại bước 3: Không có enrollments hoặc scores
  - Hiển thị empty state:
    - "Bạn chưa có dữ liệu học tập"
    - Button "Browse Courses"
  - Kết thúc Use Case

- **AF2: Student filter theo khoảng thời gian**

  - Tại bước 5: Student chọn time range
  - Dropdown options:
    - "Tất cả thời gian" (mặc định)
    - "30 ngày qua"
    - "3 tháng qua"
    - "6 tháng qua"
    - "Năm nay"
    - "Tùy chỉnh" (date range picker)
  - Hệ thống reload report với filter
  - Chart và stats cập nhật theo range
  - Kết thúc Use Case

- **AF3: Student click vào course trong table**

  - Tại bước 5B: Student click vào tên course
  - Hệ thống hiển thị detailed performance cho course đó:
    - Danh sách tất cả quizzes/assignments
    - Điểm từng bài
    - Attempts (nếu có nhiều lần thử)
    - Time spent trên từng lesson
    - Progress breakdown theo sections
  - Kết thúc Use Case

- **AF4: Student xem chi tiết điểm yếu**

  - Tại bước 5D: Student click vào weakness topic
  - Hệ thống hiển thị:
    - Các quizzes/assignments liên quan có điểm thấp
    - Lessons recommended để ôn tập
    - Related courses để cải thiện
  - Button "Study Now" → navigate đến lesson/course
  - Kết thúc Use Case

- **AF5: Student click vào recommended course**

  - Tại bước 5E: Student click vào course gợi ý
  - Chuyển hướng đến course detail page
  - Student có thể enroll hoặc xem thêm thông tin
  - Kết thúc Use Case

- **AF6: Student export report**

  - Tại bước 6: Student click "Export Report"
  - Hệ thống hiển thị dialog chọn format:
    - PDF (formatted report)
    - CSV (raw data)
  - Instructor chọn format
  - Hệ thống generate file:
    - **PDF:** Report đẹp với charts, logo
    - **CSV:** Raw data table
  - Download: "Performance_Report\_[Student_Name]\_[Date].pdf"
  - Kết thúc Use Case

- **AF7: Student so sánh với trung bình**

  - Tại bước 5: Toggle "So sánh với TB lớp"
  - Hệ thống fetch average scores của tất cả students trong cùng courses
  - Hiển thị thêm column "TB lớp" trong table
  - Chart hiển thị thêm line "Class Average"
  - Student thấy performance của mình so với peers
  - Kết thúc Use Case

- **AF8: Không có điểm quiz/assignment**
  - Tại bước 4: Student chỉ xem videos, chưa làm quiz/assignment
  - Điểm TB = N/A hoặc "-"
  - Hiển thị message: "Bạn chưa hoàn thành quiz/assignment nào"
  - Gợi ý: "Làm quiz để đánh giá kiến thức"
  - Kết thúc Use Case
