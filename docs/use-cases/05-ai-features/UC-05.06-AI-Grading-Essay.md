# Use Case-05.06: AI Grading Cho Essay Assignment

**Use Case ID:** UC-05.06

**Module:** Tính Năng AI (AI Features)

**Actor chính:** Hệ thống (System), Giảng viên (Instructor)

**Actor phụ:** Gemini AI API, Học viên (Student - passive)

**Mô tả ngắn:** AI chấm essay dựa trên rubric điểm giảng viên cung cấp

**Trigger:** Student submit essay assignment có enable AI Grading

**Pre-condition:**

- Assignment type = Essay
- Instructor đã cung cấp Rubric (required)
- Instructor đã enable checkbox "AI Grading"
- Max score đã được thiết lập
- Student đã submit essay

**Post-condition:**

- Essay được AI chấm điểm
- AI score và feedback được lưu vào database
- Student nhận thông báo có điểm
- Nếu confidence thấp: instructor nhận notification cần review

**Main Flow:**

1. Student submit essay assignment (text hoặc upload document)
2. System lưu submission vào database
3. System kiểm tra assignment có enable_ai_grading = true
4. System trigger AI grading process
5. System chuẩn bị data gửi Gemini API:

   - Question/prompt của assignment
   - Student answer (essay text)
   - Rubric (grading criteria)
   - Max score

6. System gọi Gemini API với prompt:

   ```
   Grade this essay based on the rubric.
   Question: [assignment_question]
   Student Answer: [essay_text]
   Rubric: [rubric_text]
   Max Score: [max_score]

   Return JSON:
   {
     "score": <number>,
     "feedback": "<string>",
     "strengths": ["<string>"],
     "improvements": ["<string>"],
     "confidence": <0-100>
   }
   ```

7. Gemini API process và trả về response
8. System parse AI response
9. System validate response:

   - Score trong khoảng 0 đến max_score
   - Feedback non-empty
   - Confidence là số 0-100

10. System save vào database (submission table):

    - ai_score
    - ai_feedback
    - ai_strengths (JSON array)
    - ai_improvements (JSON array)
    - ai_confidence
    - graded_at
    - grading_method = "AI"

11. System kiểm tra confidence level:

    - Nếu ai_confidence < 70: set needs_manual_review = true

12. Nếu needs_manual_review = true:

    - Send notification đến instructor
    - Subject: "Cần review: Essay có độ tin cậy thấp"
    - Include: student name, assignment title, confidence score

13. System send notification đến student:

    - "Bài essay của bạn đã được chấm"
    - Hiển thị AI score với badge "AI Graded"

14. Student view submission:

    - Hiển thị AI Score với badge "🤖 AI Graded"
    - Hiển thị AI Feedback
    - Hiển thị Strengths và Improvements
    - Disclaimer: "This is an AI-generated score. Your instructor may review and adjust."

15. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Assignment không có Rubric**

  - Tại bước 3: Rubric = null hoặc empty
  - System không trigger AI grading
  - Set grading_method = "Manual"
  - Instructor phải chấm thủ công
  - Kết thúc Use Case

- **AF2: AI Grading không được enable**

  - Tại bước 3: enable_ai_grading = false
  - System skip AI grading
  - Set grading_method = "Manual"
  - Kết thúc Use Case

- **AF3: Gemini API timeout**

  - Tại bước 7: API không response trong 15 giây
  - System retry 3 lần với exponential backoff (5s, 10s, 15s)
  - Nếu vẫn fail:
    - Set grading_status = "failed"
    - Set needs_manual_review = true
    - Send notification đến instructor
    - Student thấy "Pending grading"

- **AF4: AI response invalid**

  - Tại bước 9: Score ngoài range hoặc missing fields
  - System retry 1 lần
  - Nếu vẫn invalid:
    - Set grading_status = "failed"
    - Set needs_manual_review = true
    - Notify instructor

- **AF5: Confidence rất thấp (<50)**

  - Tại bước 11: ai_confidence < 50
  - Set needs_manual_review = true
  - Send urgent notification đến instructor
  - Student thấy score nhưng có warning: "Score may be adjusted by instructor"

- **AF6: Rate limit reached**
  - Tại bước 6: System đã vượt 1000 AI grading requests/day
  - Queue submission để grade sau
  - Student thấy "Grading in progress..."
  - Process khi rate limit reset (midnight)
