# Use Case-05.01: Upload Giáo Trình

**Use Case ID:** UC-05.01

**Module:** Tính Năng AI (AI Features)

**Actor chính:** Giảng viên (Instructor)

**Actor phụ:** Cloud Storage Service

**Mô tả ngắn:** Upload tài liệu giáo trình để làm nguồn cho AI tạo quiz

**Trigger:** Giảng viên nhấn "Upload Materials" trong section hoặc course

**Pre-condition:**

- Giảng viên đã đăng nhập
- Giảng viên sở hữu khóa học

**Post-condition:**

- Tài liệu được upload lên cloud storage
- Text được extract và lưu vào database
- Tài liệu sẵn sàng để generate quiz

**Main Flow:**

1. Giảng viên navigate đến Section hoặc Course Materials
2. Giảng viên nhấn button "Upload Materials"
3. Hệ thống hiển thị upload interface với drag-drop area
4. Giảng viên chọn file hoặc drag-drop file vào area
5. Hệ thống validate file:

   - Check định dạng: PDF (with text layer) hoặc DOCX
   - Check kích thước: ≤ 50MB
   - Nếu PDF: check có text layer (không phải scan image)

6. Hệ thống hiển thị tiến trình upload (progress bar 0-100%)
7. Hệ thống upload file lên cloud storage (S3)
8. Khi upload hoàn tất, hệ thống extract text:

   - PDF: sử dụng PDF parser library
   - DOCX: sử dụng DOCX parser

9. Hệ thống lưu vào database:

   - document_id (UUID)
   - file_name
   - file_type (PDF/DOCX)
   - file_size
   - file_url (cloud storage URL)
   - extracted_text
   - uploaded_by (instructor_id)
   - uploaded_at

10. Hệ thống hiển thị success message với preview first 500 characters
11. Hệ thống hiển thị tài liệu trong danh sách materials
12. Kết thúc Use Case

**Alternate Flow:**

- **AF1: File quá lớn**

  - Tại bước 5: File size > 50MB
  - Hệ thống hiển thị error: "File quá lớn. Kích thước tối đa 50MB."
  - Quay lại bước 3

- **AF2: Định dạng không hỗ trợ**

  - Tại bước 5: File không phải PDF hoặc DOCX
  - Hệ thống hiển thị error: "Định dạng không hỗ trợ. Vui lòng upload PDF hoặc DOCX."
  - Quay lại bước 3

- **AF3: PDF không có text layer (scanned image)**

  - Tại bước 5: PDF chỉ chứa hình ảnh, không có text
  - Hệ thống hiển thị error: "PDF này không có text layer. Vui lòng upload PDF có text hoặc sử dụng OCR."
  - Quay lại bước 3

- **AF4: Upload thất bại**

  - Tại bước 7: Lỗi kết nối cloud storage
  - Hệ thống hiển thị error: "Upload thất bại. Vui lòng thử lại."
  - Hiển thị button "Retry"
  - Quay lại bước 7

- **AF5: Text extraction thất bại**
  - Tại bước 8: Không extract được text (file corrupt)
  - Hệ thống hiển thị warning: "Không thể đọc nội dung file. File có thể bị lỗi."
  - Lưu file nhưng đánh dấu extraction_status = "failed"
  - File vẫn có thể download nhưng không dùng để generate quiz
