# Use Case-05.04: AI Tạo Quiz Cho Assignment

**Use Case ID:** UC-05.04

**Module:** Tính Năng AI (AI Features)

**Actor chính:** Giảng viên (Instructor)

**Actor phụ:** Gemini AI API

**Mô tả ngắn:** Tạo quiz questions từ giáo trình tổng cho assignment

**Trigger:** Instructor chọn "Generate Questions from Course Materials" khi tạo Quiz Assignment

**Pre-condition:**

- Instructor đang tạo Assignment với type = Quiz
- Khóa học đã có ít nhất 1 document được upload
- Documents có extracted_text

**Post-condition:**

- Quiz questions được generate từ multiple documents
- Questions được save trực tiếp vào assignment quiz
- Assignment sẵn sàng để publish

**Main Flow:**

1. Instructor tạo Assignment và chọn type = "Quiz"
2. Instructor nhấn button "Generate Questions from Course Materials"
3. Hệ thống hiển thị configuration dialog:

   - Multiple select: chọn source documents
     - Hiển thị tất cả documents đã upload trong course
     - Checkbox interface
     - Có thể chọn nhiều documents
   - Input: Số câu hỏi tổng (5-50, default 10)
   - Sliders: Phân bố độ khó (Dễ:Trung:Khó, default 30:50:20)
   - Textarea: Topic focus (optional, max 200 chars)

4. Instructor cấu hình và nhấn "Generate"
5. Hệ thống hiển thị loading indicator
6. Backend combine extracted_text từ tất cả documents đã chọn
7. Backend kiểm tra total text length:

   - Nếu > 10,000 tokens: smart chunking
     - Chia text thành chunks hợp lý
     - Phân bổ số câu hỏi đều cho mỗi chunk

8. Backend gọi Gemini API:

   - Nếu text ngắn: 1 API call với full text
   - Nếu text dài: multiple API calls cho từng chunk

9. Gemini API trả về questions
10. Backend merge questions từ các chunks (nếu có)
11. Backend validate questions (UC-05.05)
12. Hệ thống hiển thị preview table tương tự UC-05.03:

    - Question, Difficulty, Correct Answer, Actions
    - Editable inline

13. Instructor có các actions:

    - Edit/Delete questions
    - Regenerate all
    - Add more questions
    - Save to Assignment

14. Instructor nhấn "Save to Assignment"
15. Hệ thống save questions trực tiếp vào assignment_quiz
16. Hệ thống hiển thị success: "Đã thêm X câu hỏi vào assignment"
17. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Không có document nào trong course**

  - Tại bước 3: Không có document để chọn
  - Hệ thống hiển thị message: "Vui lòng upload tài liệu trước."
  - Hiển thị button "Upload Materials"
  - Kết thúc Use Case

- **AF2: Total text quá dài (>10k tokens)**

  - Tại bước 7: Text length vượt ngưỡng
  - Hệ thống tự động smart chunking:
    - Chia theo sections/chapters logical
    - Mỗi chunk ~3000 tokens
  - Hiển thị progress: "Generating questions from chunk 1/3..."
  - Continue với bước 8

- **AF3: API timeout trên một chunk**

  - Tại bước 8: 1 trong các API calls timeout
  - Backend retry chunk đó (max 3 times)
  - Nếu vẫn fail: skip chunk, tiếp tục các chunks khác
  - Hiển thị warning: "Không thể generate từ một số nội dung. Đã generate X câu hỏi."

- **AF4: Instructor muốn thêm câu hỏi thủ công**

  - Tại bước 13: Instructor nhấn "Add Manual Question"
  - Hệ thống hiển thị question editor form
  - Instructor nhập question, options, correct answer
  - Question được thêm vào preview table
  - Continue với bước 13

- **AF5: Instructor nhấn "Regenerate All"**

  - Tại bước 13: Instructor không hài lòng
  - Confirm dialog: "Regenerate tất cả?"
  - Nếu Yes: quay lại bước 5 (giữ document selection và config)

- **AF6: Rate limit reached**
  - Tại bước 8: Instructor đã generate >20 lần trong ngày
  - Hệ thống hiển thị error: "Bạn đã đạt giới hạn 20 lần generate/ngày. Vui lòng thử lại ngày mai."
  - Kết thúc Use Case
