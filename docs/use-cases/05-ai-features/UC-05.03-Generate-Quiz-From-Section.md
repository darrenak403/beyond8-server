# Use Case-05.03: AI Tạo Quiz Từ Giáo Trình Section

**Use Case ID:** UC-05.03

**Module:** Tính Năng AI (AI Features)

**Actor chính:** Giảng viên (Instructor)

**Actor phụ:** Gemini AI API

**Mô tả ngắn:** Tạo quiz questions từ tài liệu giáo trình của section/chapter

**Trigger:** Instructor nhấn "Generate Quiz from Materials" trong section

**Pre-condition:**

- Section đã có ít nhất 1 document được upload
- Document có extracted_text trong database
- Instructor có quyền chỉnh sửa khóa học

**Post-condition:**

- Quiz questions được generate bởi AI
- Questions hiển thị trong preview table
- Instructor có thể edit, delete, hoặc save questions

**Main Flow:**

1. Instructor navigate đến Section
2. Instructor nhấn button "Generate Quiz from Materials"
3. Hệ thống hiển thị configuration dialog:

   - Dropdown: chọn source document (danh sách documents đã upload trong section)
   - Input: Số câu hỏi (5-50, default 10)
   - Sliders: Phân bố độ khó (Dễ:Trung:Khó, default 30:50:20)
   - Textarea: Topic focus (optional keywords, max 200 chars)

4. Instructor cấu hình và nhấn "Generate"
5. Hệ thống hiển thị loading indicator "Generating questions..."
6. Backend lấy extracted_text từ document đã chọn
7. Backend gọi Gemini API với prompt:

   - Document text
   - Số câu hỏi yêu cầu
   - Độ khó ratio
   - Topic keywords (nếu có)

8. Gemini API trả về JSON array of questions
9. Backend validate questions (UC-05.05)
10. Hệ thống hiển thị preview table với columns:

    - Question text
    - Difficulty (Easy/Medium/Hard)
    - Correct Answer
    - Actions (Edit, Delete)

11. Each row editable inline:

    - Click vào cell để edit
    - Auto-save khi blur

12. Instructor có các actions:

    - Edit question/options inline
    - Delete individual questions
    - Button "Regenerate All"
    - Button "Add to Quiz Bank"
    - Button "Create Quiz Now"

13. Questions được đánh dấu metadata:

    - generated_by = "AI"
    - source_document_id
    - generated_at

14. Kết thúc Use Case

**Alternate Flow:**

- **AF1: Không có document trong section**

  - Tại bước 3: Section chưa có document nào
  - Hệ thống hiển thị message: "Vui lòng upload tài liệu trước khi generate quiz."
  - Hiển thị button "Upload Materials"
  - Kết thúc Use Case

- **AF2: Gemini API timeout**

  - Tại bước 8: API không response trong 60 giây
  - Hệ thống hiển thị error: "Timeout. Vui lòng thử lại."
  - Kết thúc Use Case

- **AF3: Gemini API error**

  - Tại bước 8: API trả về error
  - Backend retry 3 lần với exponential backoff
  - Nếu vẫn fail: hiển thị error "Không thể generate questions. Vui lòng thử lại sau."
  - Kết thúc Use Case

- **AF4: Validation failed**

  - Tại bước 9: Một số questions không pass validation
  - Hệ thống tự động regenerate failed questions (max 3 retries)
  - Nếu vẫn fail: chỉ hiển thị questions passed validation
  - Hiển thị warning: "Chỉ generate được X/Y câu hỏi."

- **AF5: Instructor nhấn "Regenerate All"**

  - Tại bước 12: Instructor không hài lòng với kết quả
  - Hệ thống confirm: "Regenerate tất cả câu hỏi?"
  - Nếu Yes: quay lại bước 5 (giữ nguyên cấu hình)

- **AF6: Instructor nhấn "Add to Quiz Bank"**

  - Tại bước 12: Instructor muốn lưu vào quiz bank
  - Hệ thống save questions vào QuizBank table
  - Hiển thị success: "Đã thêm X câu hỏi vào Quiz Bank"

- **AF7: Instructor nhấn "Create Quiz Now"**
  - Tại bước 12: Instructor muốn tạo quiz ngay
  - Hệ thống chuyển đến Quiz creation form
  - Questions được pre-fill sẵn
  - Instructor nhập quiz name, time limit, passing score
  - Lưu quiz vào section
